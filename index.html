iview for Taberareloo

  • Forked from iview for Tombloo
  • Works as a patch for Taberareloo
  • Uses wedata@GitHub/iview for SITEINFOs

How to Setup

  1. Install Taberareloo
  2. Install iview for Taberareloo
    • Open the above link
    • Right-click on the page
    • Select 'Patch - Install this' on the Taberareloo menu
  3. Reload this page

Cf.) https://github.com/taberareloo/iview-for-taberareloo

How to Use

  1. Select an image source on the following list
  2. Navigate images
    • J for the next image
    • K for the previous image
    • T to share using Taberareloo (+SHIFT to show QuickPostForm)
    • O to open the original page where an image comes from
    • B to go back home (to avoid reloading SITEINFOs)
